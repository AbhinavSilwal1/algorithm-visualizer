# Algorithm Visualizer

A Python-based sorting algorithm visualizer using Tkinter, generators, and real-time animation.

## Current Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

## Features
- Real-time sorting visualization
- Multiple sorting algorithm animations
- Color-coded dynamic visual feedback
- Live comparison and swap counters
- Random array generation
- Interactive Tkinter GUI
- Dynamic algorithm legend display
- Algorithm selection dropdown menu

## Technologies Used
- Python
- Tkinter
